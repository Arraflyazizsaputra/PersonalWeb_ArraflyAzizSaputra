Hallo....!!!!!!!
Selamat datang di repository github saya (ARRAFLY AZIZ SAPUTRA) PersonalWeb ini di buat untuk memenuhi tugas Dasar Pemorograman semester II sebagai mahasiswa Fasilkom Universitas Subang, PersonalWEB ini dibuat menggunakan TaillwindCSS cdn 4.0 dengan HTML specialCharacter, Berikut Deskripsi mengenai isi dari fitur-fitur Web Buatan Saya

1. ![Screenshot 2025-06-23 111850](https://github.com/user-attachments/assets/255731e8-837a-4748-abf4-92e807908a7a)
    Tampilan dan fitur web HTML dan PHP ini membentuk sebuah halaman beranda (home) dari website personal bernama "Personal Web | ARRAFLY AZIZ SAPUTRA" yang didesain secara modern, interaktif, dan responsif menggunakan **Tailwind CSS**. Pada bagian atas halaman terdapat **header mewah** dengan gradasi warna biru hingga cyan, ikon animasi 🌐✨, dan judul besar berwarna putih dengan highlight kuning yang menunjukkan nama pemilik web. Di bawah header, terdapat **navigasi menu yang elegan** dengan efek **glassmorphism**, responsif untuk tampilan mobile dan desktop. Pada tampilan mobile, menu akan berubah menjadi dropdown yang bisa ditoggle menggunakan tombol hamburger.
    Selanjutnya, terdapat **form pencarian artikel** yang dirancang menarik, berwarna biru gelap dengan input bergaya modern dan ikon pencarian, memungkinkan pengguna untuk mencari artikel berdasarkan judul atau isi. Konten utama terbagi dua kolom: **bagian artikel utama** dan **sidebar**. Artikel utama menampilkan daftar artikel terbaru dengan judul, ringkasan 200 karakter, dan tautan menuju halaman detail. Setiap artikel memiliki ID unik agar bisa dirujuk dari sidebar. Sementara itu, **sidebar** berisi daftar judul artikel dengan emoji lucu seperti 📝, 📘, 📄, dan lainnya yang dapat diklik untuk men-scroll secara halus ke artikel terkait di bagian utama, lengkap dengan efek **highlight sementara** saat diklik. Sidebar ini juga memuat **fitur kalender interaktif** yang menampilkan hari dan tanggal pada bulan berjalan, serta menandai tanggal saat ini dengan warna kuning cerah.
    Fitur desain lainnya mencakup **dark mode** yang aktif dengan class Tailwind `dark:`, efek hover yang menyenangkan, serta penggunaan berbagai elemen UI seperti bayangan (`shadow`), transisi halus (`transition duration-300`), dan elemen interaktif seperti animasi bounce dan pulse. Footer halaman menampilkan informasi copyright dinamis yang menyesuaikan dengan tahun saat ini. Secara keseluruhan, halaman ini dirancang tidak hanya fungsional dan mudah digunakan, tetapi juga estetis dan menyenangkan untuk dijelajahi oleh pengunjung.

4. ![Screenshot 2025-06-23 112100](https://github.com/user-attachments/assets/87cb2e12-7e27-4d78-84c7-52fc28cd715a)
    halaman "Tanya Admin" ini memiliki desain modern, responsif, dan memanfaatkan efek gradasi warna biru dengan sentuhan glassmorphism pada header dan navigasi. Pada bagian atas, terdapat header yang menampilkan judul halaman disertai dengan ikon lucu "🌐✨" dan nama pemilik situs dengan penekanan pada warna kuning. Navigasi utama disusun horizontal untuk layar besar dan berubah menjadi menu hamburger responsif untuk tampilan mobile, yang dapat ditoggle untuk menampilkan daftar menu seperti Artikel, Gallery, About, Tanya Admin, Data Pengunjung, dan tombol login berwarna mencolok.
    Bagian utama halaman menampilkan form untuk mengirim pertanyaan kepada admin. Form ini dirancang dalam card penuh warna dengan latar gradasi dari biru tua ke cyan, bayangan lembut, sudut membulat, dan teks berwarna putih. Data nama dan pesan dari form akan disanitasi menggunakan mysqli_real_escape_string dan jika tidak kosong, akan disimpan ke dalam tabel tbl_pesan. Jika berhasil, akan muncul notifikasi alert bahwa pesan berhasil dikirim, jika tidak, akan muncul pesan kesalahan. a. Pemeriksaan sesi login: Jika $_SESSION['username'] sudah ada, pengguna akan otomatis diarahkan ke beranda_admin.php, mencegah login ulang.
